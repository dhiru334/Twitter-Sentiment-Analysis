# Twitter Sentiment Analysis

## Project Overview
This project focuses on analyzing Twitter data to classify sentiments into positive, negative, and neutral categories using machine learning techniques.

The project uses the Naive Bayes algorithm along with CountVectorizer for text feature extraction. The model was trained and evaluated using a Twitter dataset and achieved approximately 81% accuracy.


## Objective
- Analyze Twitter text data
- Classify tweets into sentiments
- Understand public opinion using machine learning


## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn


## Dataset
The dataset contains tweets and sentiment labels:
- Positive
- Negative
- Neutral


## Project Workflow
1. Data Loading
2. Data Exploration
3. Data Preprocessing
4. Feature Extraction using CountVectorizer
5. Train-Test Split
6. Model Training using Naive Bayes
7. Model Evaluation
8. Visualization
9. Custom Sentiment Prediction


## Results
- Model Accuracy: **81%**
- Successfully classified tweet sentiments
- Visualized sentiment distribution using graphs
